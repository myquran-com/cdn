CDN MyQuran.com

Content Delivery Network untuk Gambar Ayat dan Halaman Al-Qur'an

Tentang CDN Ini

CDN ini menyediakan akses cepat ke file gambar ayat-ayat Al-Qur'an dan halaman-halaman Al-Qur'an dalam format PNG. Ini adalah bagian dari proyek MyQuran.com untuk menyediakan konten Al-Qur'an dalam format digital yang mudah diakses.

Struktur Direktori

  • /ayah/ - Berisi file gambar ayat Al-Qur'an
  • /page/ - Berisi file gambar halaman Al-Qur'an

Format Nama File

  • Ayat: [nomor_surah]_[nomor_ayat].png
  • Halaman: [nomor_halaman].png

Contoh URL

https://cdn.myquran.com/ayah/1_1.png
https://cdn.myquran.com/ayah/114_6.png
https://cdn.myquran.com/page/1.png
https://cdn.myquran.com/page/604.png

Penggunaan

Gunakan URL di atas untuk mengintegrasikan gambar ayat Al-Qur'an ke dalam aplikasi atau situs Anda:

<img src="https://cdn.myquran.com/ayah/1_1.png" alt="Al-Fatihah 1">

Lisensi

Konten dalam CDN ini adalah bagian dari sumber daya Al-Qur'an digital. Harap gunakan secara bertanggung jawab dan hormati hak cipta yang berlaku.
