# CDN MyQuran.com

Repository ini berisi file-image yang digunakan sebagai CDN untuk domain `cdn.myquran.com`. Gambar-gambar ini digunakan untuk menyajikan ayat-ayat Al-Qur'an dalam bentuk gambar (image) dan halaman-halaman Al-Qur'an.

## Struktur Direktori

```
.
├── ayah/
│   └── [nomor_surah]_[nomor_ayat].png
├── page/
│   └── [nomor_halaman].png
└── README.md
```

- Direktori `ayah/` berisi gambar-gambar ayat Al-Qur'an dengan format nama file `[nomor_surah]_[nomor_ayat].png`
- Direktori `page/` berisi gambar-gambar halaman Al-Qur'an dengan format nama file `[nomor_halaman].png`
